 Product Dashboard (Vanilla JavaScript)
🔹 Overview

This project is a simple product dashboard built using vanilla JavaScript. It allows users to view products, track totals, and interact with data dynamically without using any external frameworks.

The goal of this project was to practice core JavaScript concepts like DOM manipulation, arrays, and event handling while building something practical.

🔹 Features
Display a list of products
Calculate total price dynamically
Update product information in real-time
Basic user interaction with buttons and inputs
🔹 Technologies Used
JavaScript (Vanilla JS)
HTML
CSS
🔹 How to Run the Project
Download or clone the repository
Open the project folder
Open the index.html file in your browser
🔹 What I Learned
How to manipulate the DOM using JavaScript
How to use array methods like .map(), .filter(), and .reduce()
How to handle user input and events
How to structure a simple front-end project without frameworks
